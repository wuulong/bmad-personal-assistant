# 執行研究 (Conduct Research)

## Purpose

此任務旨在針對特定主題進行全面的資訊搜尋和收集，為使用者提供深入、全面的研究報告。

## Inputs

- **`research_topic`**: 需要進行研究的具體主題或問題。
- **`scope`** (Optional): 研究的範圍和深度要求（例如：廣泛、深入、特定領域）。
- **`source_preference`** (Optional): 偏好的資訊來源（例如：學術論文、新聞報導、市場報告、網路論壇）。
- **`time_constraint`** (Optional): 研究的時間限制。

## Process

### 1. 定義研究問題與關鍵詞

- 將 `research_topic` 轉化為明確的研究問題。
- 根據研究問題，生成相關的關鍵詞和搜尋詞組。

### 2. 搜尋與資訊收集

- 根據 `source_preference`，在指定或推薦的資訊來源中進行搜尋。
- 收集與研究主題相關的文獻、數據、報告、文章等。
- 記錄資訊來源，確保可追溯性。

### 3. 篩選與評估資訊

- 根據 `scope` 和資訊的可靠性、相關性，篩選收集到的資訊。
- 評估資訊的權威性、時效性和客觀性。

### 4. 整理與分類資訊

- 將篩選後的資訊進行分類、歸納和整理。
- 建立清晰的資訊結構，方便後續分析。

## Outputs

- **`research_materials`**: 收集到的原始研究材料清單（包含來源連結或引用）。
- **`categorized_information`**: 整理和分類後的資訊結構。
- **`preliminary_summary`** (Optional): 對收集到的資訊進行初步總結。

## Key Principles

- **全面性**: 盡可能涵蓋研究主題的各個方面。
- **準確性**: 確保收集到的資訊真實可靠。
- **客觀性**: 避免在收集過程中帶入個人偏見。
- **效率**: 在時間約束內高效完成資訊收集。
